Merci d'avoir rejoins le projet ! (Tu peux essayer de zoomer si tes yeux te font mal).
Pour contribuer au projet, chacun créera son module qui sera ensuite accessible via un menu.
Pour le moment, quelques fonctions et classes "basiques" sont disponibles, comme celle qui affiche ce texte de manière "naturelle", ou le petit label que tu as vu en haut à droite.
Le temps de dévelloppement est assez rapide, tous les fichiers sources accessibles dans ce dépot ont été créé en quelques jours!
N'hésite pas à te laisser aller et proposer tes meilleures idées. Si tu penses ne pas être capable d'implémenter ton idée, faute de temps ou de moyens, n'hésite pas à en parler aux autres membre du groupe,
on est là pour ça !

N'oublie pas d'aller relire les règles du concours avant de coder:
- https://trophees-nsi.fr/media/pages/ressources/75a3a18927-1738880572/trophees-nsi-reglement-2025-vdef.pdf

Plus d'infos seront disponibles en présentielle ou plus tardivement dans le dossier `docs/`

Quelques commandes git:
- `git branch nom-de-la-branche`                              <- Permet de créer une branche 
- `git checkout nom-de-la-branche`                            <- Permet de se mettre sur la branche donnée
- `git add chemin_du_fichier`                                 <- Permet d'ajouter les fichiers à enregistrer comme modifiés avant un commit
- `git commit -m "Nom du commit" -m "Message ou description"` <- Sauvegarde les changement effectué sur la branche
- `git push -u nom-de-la-branche`                             <- "Publie les changements (commit) sur la branche donnée en ligne 
- `git pull`                                                  <- Pour se mettre à jour sur la branche actuelle si des changements ont été fait
- `git stash`                                                 <- Si jamais vous avez effectué des changements non sauvegardés et que vous souhaitez aller voir une autre branche

Bon courage !!
